# 💰 Harcama Takip Uygulaması

**MS323 Betik Dilleri Dersi Final Projesi**

Bu proje, kullanıcıların günlük harcamalarını konsol üzerinden kolayca kaydetmesini, listelemesini ve toplam giderlerini takip etmesini sağlayan Python tabanlı bir terminal uygulamasıdır.

## 🚀 Özellikler

- **Harcama Ekleme:** Açıklama, tutar ve tarih (varsayılan olarak bugünün tarihi) bilgileriyle yeni harcama kaydı oluşturma.
- **Sıralı Listeleme:** Kaydedilen tüm harcamaları, tutara göre büyükten küçüğe doğru sıralı bir tablo halinde görüntüleme.
- **Toplam Tutar Hesaplama:** Yapılan toplam harcama miktarını ve sistemdeki kayıtlı işlem sayısını anlık olarak gösterme.
- **Güvenli Veri Girişi:** Kullanıcı kaynaklı hataları (boş bırakma, harf girme, geçersiz tarih formatı) engellemek için kapsamlı hata yakalama (Exception Handling) mekanizmaları.

## 🛠️ Kullanılan Teknolojiler

- **Dil:** Python 3.x
- **Modüller:** `datetime` (Tarih ve zaman işlemleri için standart kütüphane)
